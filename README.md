# Roulette Project

## Описание проекта

Этот проект создан для тестирования **Qwen 3.5**.

## Что это такое?

Это простая рулетка, где вы можете:

- ✍️ Вводить свои собственные слова
- 🎡 Крутить рулетку
- 🎲 Получать случайные слова из вашего списка

## Как использовать

1. Добавьте свои слова в список
2. Нажмите кнопку "Крутить" (или аналогичную)
3. Рулетка остановится на случайном слове из вашего списка

## Особенности

- Простой и интуитивный интерфейс
- Возможность добавлять неограниченное количество слов
- Случайный выбор слова при каждом вращении
